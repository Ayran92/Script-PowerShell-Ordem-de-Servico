# Script-PowerShell-Ordem-de-Servico
Este projeto contém um script PowerShell para automatizar a geração e organização de Ordens de Serviço. O script foi criado para facilitar rotinas internas, evitando tarefas repetitivas e garantindo padronização no registro das informações.

Principais funções (modelo):

Coleta de dados do cliente e do serviço executado.

Geração automática do documento de ordem de serviço.

Organização dos arquivos por data ou número.

Registro rápido e padronizado das informações.
